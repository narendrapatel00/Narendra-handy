# Narendra-handy

#Deploy link : https://bright-choux-82e1d5.netlify.app/

Narendra Handy is a zero-dependency, single-file browser app that turns your webcam into a gesture-controlled augmented reality canvas. 
 Built on Google's MediaPipe Hands, it tracks 21 hand landmarks in real time and maps them to visual effects, physics simulations, and freehand drawing — all rendered on layered HTML5 canvases with a neon matrix aesthetic. Three interaction modes:

Play — Pinch to fire shockwaves, bring both hands together for lightning arcs and a rotating mandala Shapes — Form a heart with both hands, flash a peace sign, or close both fists for a star burst Write — Point your index finger to draw glowing paths on the canvas; pinch to lift the pen; shake an open palm to clear

No backend. No build step. No npm. Just open the HTML file in Chrome or Edge, allow camera access, and start gesturing.
